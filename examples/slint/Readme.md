# Slint Loves MapLibre

This example shows how MapLibre can be used in Slint. This is a prove of concept right now and quite inefficient because it copies the image to the ram instead of keeping it on the gpu and using it from there in slint. In the follow up [PR](https://github.com/maplibre/maplibre-native-rs/pull/143) this problem shall be solved as well
